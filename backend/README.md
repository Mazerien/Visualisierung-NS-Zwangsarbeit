# Backend
## Setup
Create virtual environment, install requirements with `pip install -r requirements.txt`  
Start with `python app.py`.  
Check if it works by going to: `localhost:5000/api/hello`  

## Directory structure
```
backend/
├── app.py
├── osm.py
└── api/

    ├── osm.py

```
